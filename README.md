# The Hustle OS v1

A comprehensive business management system with AI-driven analytics, RAG (Retrieval Augmented Generation), and voice integration.

## 📁 Repository Structure
- `/frontend`: Vite + React UI
- `/backend`: Node.js + Express API
- `/Auto-LLM/analytics-llm`: Python + Flask AI & RAG Engine

## 🚀 Deployment
For detailed instructions on deploying this system to a Hostinger VPS or any Linux server, please refer to the **[DEPLOYMENT.md](./DEPLOYMENT.md)** guide.

## 🛠 Local Development
1. Start the Node backend: `cd backend && npm run dev`
2. Start the Python AI API: `cd Auto-LLM/analytics-llm && python3 api.py`
3. Start the Vite frontend: `cd frontend && npm run dev`

## 🎙️ AI Features
This system supports:
- **RAG Chat**: Ask questions about your business data in natural language.
- **AI Insights**: Automated performance and project tracking.
